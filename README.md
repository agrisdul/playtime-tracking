Create new project folder for example: "pinball_timer"
Create or copy 3 files:
server.py
admin.html
screen.html

After that Run command: python3 server.py 
App runs on port 8080.


Step D — Open it on iPad + TV
Find the laptop’s local IP address (example 192.168.1.50).
On the iPad, open:
http://192.168.1.50:8080/admin.html
On the TV device browser (or laptop connected via HDMI), open:
http://192.168.1.50:8080/screen.html
Now add “JOEL / 60 min” on iPad → JOEL instantly appears on TV and counts down
